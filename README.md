### Setup

```bash
$ cd backend
$ yarn install
$ npx knex migrate:latest

$ cd frontend
$ yarn install

$ cd mobile
$ yarn install
```

### Run the application

```bash
$ cd backend 
$ yarn start

$ cd frontend
$ yarn start

$ cd mobile
$ yarn start
```
